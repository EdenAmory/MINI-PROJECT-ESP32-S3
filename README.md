## Komponen I/O yang Diuji

1.  **LED** (Output) - Pin 2
2.  **BUZZER** (Output) - Pin 10
3.  **BUTTON OPEN** (Input) - Pin 35
4.  **BUTTON CLOSE** (Input) - Pin 14
5.  **POTENSIOMETER (POT)** (Input Analog) - Pin 5
6.  **OLED Display** (Output) - Pin 38 (SDA), 39 (SCL)
8.  **SERVO Motor** (Output) - Pin 18
